TaxBridge<br><br>
TaxBridge is an automation tool that extracts transactions from bank statements, classifies expenses, applies GST rules, and generates ready to download reports.<br><br>
<br>
Features<br>
Upload statements in PDF, CSV, or Excel format<br>
Extract and clean transaction data<br>
Auto-generate category, GST rate, and GST input<br>
Provide basic spend and GST summaries<br>
Export results as Excel or CSV<br><br>
<br>
Tech Stack<br>
Python<br>
Streamlit<br>
Pandas<br><br>
